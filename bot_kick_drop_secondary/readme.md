# Description | 內容
Survivor bots will drop their secondary weapon when they were kicked

* Apply to | 適用於
	```
	L4D1
	L4D2
	```

* Image | 圖示
	| Before (裝此插件之前)  			| After (裝此插件之後) |
	| -------------|:-----------------:|
	| ![bot_kick_drop_secondary_before_1](image/bot_kick_drop_secondary_before_1.gif)|![bot_kick_drop_secondary_after_1](image/bot_kick_drop_secondary_after_1.gif)|

* <details><summary>How does it work?</summary>

	* (Before) When survivor bots were kicked by server or plugins, they will not drop their secondary weapon (disappear), only drop primary weapons, throwables, medical items
	* (After) When survivor bots were kicked by server or plugins, they will also drop their secondary weapon
		* Pistol and Dual pistol
		* Magnum
		* Melee weapons (support custom map melee)
		* Chainsaw
</details>

* Require | 必要安裝
	1. [left4dhooks](https://forums.alliedmods.net/showthread.php?t=321696)

* <details><summary>Changelog | 版本日誌</summary>

	* v１.0 (2025-11-8)
		* Initial Release
</details>

# 中文說明
當倖存者bot被踢出遊戲時，掉落副武器

* 原理
	* (裝此插件之前) 倖存者bot被踢出遊戲時，手上副武器不會掉落直接消失, 只會掉落主武器、投擲物、醫療物品
	* (裝此插件之後) 倖存者bot被踢出遊戲時，手上副武器也會掉落
		* 掉手槍、近戰武器、麥格農手槍、電鋸
		* 可掉雙手槍、三方圖自製近戰武器
