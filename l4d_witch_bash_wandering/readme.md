# 中文說明
修復倖存者推開站立的witch不會導致驚嚇

* 原理
    * (裝此插件之前) 倖存者使用右鍵推開站立的witch，不會導致witch被驚嚇
    * (裝此插件之後) Witch會被驚嚇
